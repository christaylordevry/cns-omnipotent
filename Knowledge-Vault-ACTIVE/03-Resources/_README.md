# 03-Resources

## What Goes Here
Curated reference material, citations, research write-ups, derived insights, syntheses, and validation notes that agents can reuse across projects.

## What Does Not Go Here
Untriaged raw captures that need immediate sorting should go to `00-Inbox/`.

Project-specific workflow tracking should go to `01-Projects/` or `02-Areas/`.

Time-bound logs should go to `DailyNotes/`.

## Frontmatter Requirements
Notes in `03-Resources/` must include PAKE standard frontmatter fields:
`pake_id`, `pake_type`, `title`, `created`, `modified`, `status`, `confidence_score`, `verification_status`, `creation_method`, and `tags`.

Use the matching `pake_type` for each resource note:
- `SourceNote`, `InsightNote`, `SynthesisNote`, or `ValidationNote`.
